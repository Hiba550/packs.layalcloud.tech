Roughly Enough Items (REI)

Clean and customizable recipe viewer for Fabric

Fabric Utility by shedaniel

Description

Roughly Enough Items (REI) is a clean, customizable recipe viewer designed specifically for Fabric. It's the Fabric equivalent of JEI, offering a modern interface and extensive customization options.

Features

  • Modern, clean user interface
  • Comprehensive recipe browsing
  • Advanced search with regex support
  • Highly customizable appearance
  • Plugin system for mod integration
  • Recipe favorites and bookmarks
  • Performance optimized

Fabric Optimized

  • Built specifically for Fabric ecosystem
  • Lightweight and fast
  • Seamless integration with Fabric mods
  • Regular updates for latest Minecraft versions

Customization

  • Flexible theme system
  • Configurable layouts
  • Custom color schemes
  • Adjustable UI scaling

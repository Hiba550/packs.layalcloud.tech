Fabric API

Essential hooks and interoperability mechanisms for Fabric mods

Fabric Library by FabricMC

Description

Fabric API is the library for essential hooks and interoperability mechanisms for Fabric mods. It provides the foundational tools that most Fabric mods require to function properly.

Features

  • Essential mod compatibility layer
  • Common hooks and events system
  • Registry synchronization
  • Resource loading utilities
  • Networking capabilities
  • Cross-mod interaction support

Why You Need This

  • Required by 90% of Fabric mods
  • Enables mod compatibility
  • Provides stable modding APIs
  • Regular updates for new Minecraft versions

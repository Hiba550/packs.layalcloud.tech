Curios API

Equipment slots expansion for accessories and RPG items

Forge Library by TheIllusiveC4

Description

Curios API is an essential library mod that expands Minecraft's equipment system by adding new accessory slots. It enables rings, necklaces, charms, and other accessories to be equipped, forming the foundation for RPG-style equipment systems in modded Minecraft.

Equipment Slots

  • Ring slots for magical and utility rings
  • Necklace slots for powerful amulets and pendants
  • Bracelet slots for wearable accessories
  • Charm slots for small trinkets and tokens
  • Belt slots for utility items and tools
  • Configurable slot types and counts

Developer Features

  • Comprehensive API for mod developers
  • Easy integration with existing mods
  • Flexible slot configuration system
  • Event handling for equipment changes
  • Compatibility layer for different mod types

Popular Mods Using Curios

  • Botania - Magical baubles and accessories
  • Ars Nouveau - Magical equipment and trinkets
  • Iron's Spells - Spell-casting accessories
  • Artifacts - Unique equipment pieces
  • Many RPG and magic-themed mods

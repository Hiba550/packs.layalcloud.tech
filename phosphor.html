Phosphor

Modern lighting engine optimization for better performance

Fabric Performance by JellySquid

Description

Phosphor is a performance optimization mod that improves Minecraft's lighting engine. It dramatically reduces lighting-related lag and improves FPS, especially in areas with complex lighting calculations. Perfect companion to Sodium and Lithium for ultimate performance.

Performance Benefits

  • Optimized lighting calculations for better FPS
  • Reduced lighting-related lag and stuttering
  • Faster chunk loading in well-lit areas
  • Improved performance with many light sources
  • No visual changes to maintain vanilla appearance
  • Compatible with shader packs and OptiFine

Technical Features

  • Modern lighting engine algorithms
  • Efficient light propagation calculations
  • Optimized block light and sky light handling
  • Reduced CPU overhead for lighting updates
  • Thread-safe lighting operations

Compatibility

  • Works excellently with Sodium and Lithium
  • Compatible with most Fabric mods
  • No conflicts with visual enhancement mods
  • Supports all Minecraft versions 1.15+

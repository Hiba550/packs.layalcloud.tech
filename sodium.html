Sodium

Modern rendering engine that dramatically improves frame rates and reduces micro-stutter

Fabric Performance by CaffeineMC

Description

Sodium is a free and open-source rendering optimization mod for Minecraft 1.16+ which greatly improves frame rates and stuttering while fixing many graphical issues.

Features

  • Dramatically improved frame rates (200-400% in many cases)
  • Reduced micro-stutter and frame time spikes
  • Advanced memory management
  • Compatible with most Fabric mods
  • Extensive video settings menu
  • Modern OpenGL rendering pipeline

Requirements

  • Minecraft 1.16+ (Java Edition)
  • Fabric Loader
  • Fabric API (recommended)

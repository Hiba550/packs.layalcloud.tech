Xaero's Minimap

Fair-play minimap with waypoints, entity radar, and cave mapping

Fabric/Forge Utility by xaero96

Description

Xaero's Minimap is a fair-play minimap mod that displays the world around you as you explore. It features waypoint management, entity radar, and cave mapping while staying balanced for multiplayer servers.

Features

  • Real-time minimap display
  • Waypoint system with unlimited waypoints
  • Entity radar for mobs and players
  • Cave mapping and underground exploration
  • Customizable appearance and settings
  • Fair-play mechanics for server compatibility
  • Works great with Xaero's World Map

Fair-Play Features

  • No X-ray or cheating capabilities
  • Respects server rules and limitations
  • Configurable by server administrators
  • Balanced for competitive gameplay

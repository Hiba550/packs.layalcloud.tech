Iris Shaders

Modern shaders mod compatible with existing shader packs

Fabric Visual by IrisShaders

Description

Iris Shaders is a modern shaders mod for Minecraft that aims to be compatible with existing OptiFine shader packs while providing better performance and more features. It works seamlessly with Sodium for the ultimate visual and performance experience.

Features

  • OptiFine shader pack compatibility
  • Enhanced visual effects and lighting
  • Built-in performance optimizations
  • Seamless Sodium integration
  • Advanced shader configuration
  • Real-time shader switching
  • Custom shader development tools

Visual Enhancements

  • Dynamic lighting and shadows
  • Realistic water reflections
  • Advanced atmospheric effects
  • Improved particle systems
  • Enhanced sky rendering

Performance

  • Better performance than OptiFine shaders
  • Works with Sodium for maximum FPS
  • Efficient GPU utilization
  • Reduced CPU overhead

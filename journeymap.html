JourneyMap

Real-time mapping and waypoint system with fullscreen map and minimap

Fabric/Forge Utility by techbrew

Description

JourneyMap is a comprehensive mapping solution for Minecraft that provides real-time mapping, advanced waypoint management, and detailed exploration tracking. It's one of the most feature-rich and polished mapping mods available.

Mapping Features

  • Real-time world mapping as you explore
  • High-resolution fullscreen map
  • Customizable minimap with multiple styles
  • Underground cave mapping
  • Biome and structure detection
  • Multi-dimensional support (Nether, End)
  • Topographical and satellite view modes

Waypoint System

  • Unlimited custom waypoints
  • Waypoint categories and organization
  • Distance and direction indicators
  • Death point automatic marking
  • Shared waypoints for multiplayer
  • Import/export waypoint sets

Advanced Features

  • Mob and player tracking
  • Slime chunk detection
  • Grid overlay and coordinates
  • Day/night cycle visualization
  • Web map server for browser viewing
  • Extensive customization options

Quality of Life

  • Never get lost again
  • Track your exploration progress
  • Perfect for large modded worlds
  • Essential for server play

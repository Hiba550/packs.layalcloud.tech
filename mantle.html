Mantle

Essential library mod providing utilities and shared code for Tinkers' Construct

Forge Library by SlimeKnights

Description

Mantle is a crucial library mod developed by the SlimeKnights team that provides essential utilities, shared code, and framework components for Tinkers' Construct and other related mods. It contains the foundational systems that enable the advanced features found in Tinkers' Construct.

Features

  • Essential library for Tinkers' Construct
  • Shared utilities and common code
  • Book and manual rendering system
  • Advanced GUI components
  • Data generation tools
  • Recipe and crafting system framework
  • Material system foundation
  • Performance optimizations

Required For

  • Tinkers' Construct - Main dependency
  • Tinkers' Complement
  • Construct's Armory
  • Other SlimeKnights mods
  • Various third-party addons

Technical Features

  • Book rendering and formatting system
  • Advanced tooltip and GUI systems
  • Recipe registration framework
  • Data pack integration utilities
  • Network synchronization tools

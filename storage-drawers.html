Storage Drawers

Smart bulk storage solution with upgradeable drawers

Forge Storage by jaquadro

Description

Storage Drawers provides a compact and efficient storage solution designed specifically for bulk item storage. Perfect for organizing large quantities of materials, these smart drawers display their contents visually and support extensive automation features.

Drawer Types

  • Basic Drawer - Single slot storage
  • 1x2 Drawers - Two slots in one block
  • 2x2 Drawers - Four slots in one block
  • Half Drawers - Compact single-slot option
  • Trim blocks for decorative borders

Upgrades

  • Storage Upgrades - Iron, Gold, Diamond, Emerald tiers
  • Redstone Upgrade - Emit signal when full
  • Void Upgrade - Delete excess items automatically
  • Lock Upgrade - Prevent item type changes
  • Indicator Upgrade - Status light system

Automation Features

  • Drawer Controller - Central network management
  • Slave blocks for controller expansion
  • Hopper and pipe compatibility
  • Item insertion and extraction automation
  • Compacting Drawers for ore processing
